Detailed Description

A JSON Emitter class.

This class emits records as an array of JSON objects, as described in the csv::EmitterIface interface class.

Emit a single JSON record to an output stream.

Will write a single JSON record in the following format:

{
"<field-name-1>": <value>,
"<field-name-2>": <value>,
...
"<field-name-N>": <value>
}

The field names are retrieved from the provided specification.

The field name/value pairs are written in the same order that they appear in the vector returned by record.fields() .

String values will be quoted.

Leading commas will be added before all records except the first one.

Parameters
outputThe output file stream to emit the record to to.
specificationRecord specification to retrieve name and type from.
recordThe record to emit.
Returns
true - Record was successfully emitted.
false - Record data could not be emitted.
