Detailed Description

Called by DataSet emit records.

This class hosts a single CSV record, as provided by a ReaderIface implementation.

Host a CSV file specification with field name and type. Used when decoding CSV files and encoding the given output format. Once constructed, this object is immutable.

Convert all records from an input stream to a new format.

This function provides top-level functionality to convert records from one format to another.

It will use ingester read and parse data from input, record by record, and then emit them to output using emitter.

The format and data types of the records read is provided by specification.

The ingester is created by a call to csv::Factory<csv::IngesterIface>::produce() .

The emitter is created by a call to csv::Factory<csv::EmitterIface>::produce() .

The call flow to the emitter is like follows.

emitter.begin();
emitter.emit_record();
emitter.emit_record();
emitter.emit_record();
...
emitter.end();
Parameters
specificationThe specification of the records read from input.
ingesterThe ingestion instance to use to read data from input.
inputThe input data stream to read records from.
emitterThe emitter instance to use to write data to output
outputThe output data stream to write converted records to.
Returns
The number of records converted.

Extract fields from a single line.

This function will divide line into a vector of strings placed in result.

The line is assumed to have fields separated by the character specified by separator.

Field data can be escaped by the character specified by escape. If an escape character is encoutered, the next character will be added to the field data even if it is a separator character.

Parameters
lineThe line to separate into fields.
separatorThe separator character to use.
escapeThe escape character to use.
resultThe string vector to add fields to. \ Fields will be added after any existing elements in the vector.
Returns
The number of fields added to result.
