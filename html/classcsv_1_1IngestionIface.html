Detailed Description

An interface class to read records from an input stream.

This class can be subclassed to CSV and other input formats that are to be transformed to an output format through an csv::EmitterIface subclass.

Subclasses are managed through the csv::Factory<IngestionIFace> template instantiation, and should never be instantiated directly.

Read and parse a single record.

The implementation of this method shall:

  • Read a single record from input
  • Parse the read record according to specification
  • Return a newly created csv:Record object with the parsed data.
    Parameters
    inputThe input stream to read and parse a record data from
    specificationThe specification to use when parsing the record data.
    record_indexThe index of the current record. Incremented by one for each call to ingest_record().
    Returns
    Shared pointer to a newly created csv::Record if record was parsed.
    NULL if input has reached its end.
    Bug:
    There is no way for this function to report a failed record parsing.
