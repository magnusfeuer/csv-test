Detailed Description

A single record with parsed data.

This class stores data for a single record, ingested by a subclass to csv::IngestionIface.

The constructor parses the string tokens and converts them to internal fields with the correct datatype, as retrieved from a provided specification.

Once constructed, the Record object remains immutable for its lifetime.

For fields of type csv::FieldType::STRING, the token is stored as-is in the internal variant vector.

For fields of type csv::FieldType::INT64 and csv::FieldType::DOUBLE, the token is converted to a native int64_t or double before it is stored in the internal variant vector.
